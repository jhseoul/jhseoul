# JH Seoul — 발효의 결

전통 발효 원료로 완성한 프리미엄 스킨케어 브랜드 JH Seoul의 공식 웹사이트입니다.

## 폴더 구조
```
jhseoul-website/
├── index.html      메인 페이지
├── css/style.css   스타일
├── js/script.js    인터랙션 (스크롤 애니메이션, 뉴스레터 폼)
└── images/         이미지 폴더 (비어있음, 필요시 추가)
```

## 실행 방법
`index.html` 파일을 브라우저에서 열면 바로 확인할 수 있습니다. 별도의 설치나 빌드 과정이 필요 없습니다.
